class Ameba::Tokenizer

  • Ameba::Tokenizer
  • Reference
  • Object

Overview

Represents Crystal syntax tokenizer based on Crystal::Lexer.

source = Ameba::Source.new code, path
tokenizer = Ameba::Tokenizer.new(source)
tokenizer.run do |token|
  puts token
end
