  • Ameba::AST::NodeVisitor
  • Ameba::AST::BaseVisitor
  • Crystal::Visitor
  • Reference
  • Object

Overview

An AST Visitor that traverses the source and allows all nodes to be inspected by rules.

visitor = Ameba::AST::NodeVisitor.new(rule, source)
