class Ameba::AST::Argument

  • Ameba::AST::Argument
  • Reference
  • Object

Overview

Represents the argument of some node. Holds the reference to the variable, thus to scope.

For example, all these vars are arguments:

def method(a, b, c = 10, &block)
  3.times do |i|
  end

  ->(x : Int32) {}
end
