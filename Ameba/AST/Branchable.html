class Ameba::AST::Branchable

  • Ameba::AST::Branchable
  • Reference
  • Object

Overview

A generic entity to represent a branchable Crystal node. For example, Crystal::If, Crystal::Unless, Crystal::While are branchable.

white a > 100 # Branchable A
  if b > 2    # Branchable B
    a += 1
  end
end
